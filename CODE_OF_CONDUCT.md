# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors, maintainers, and participants are expected to uphold it.

Please read the full text at the link above.

## Reporting

If you experience or witness unacceptable behaviour, please report it by opening a private report via [GitHub](https://github.com/Yortw/winix/security/advisories/new) or contacting the maintainer directly.

All reports will be handled with discretion.
