# less Pager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a native terminal pager with GNU less compatibility, ANSI colour passthrough, search, follow mode, and modern defaults — filling the gap on Windows where no usable pager exists.

**Architecture:** Class library (`Winix.Less`) holds all logic — option resolution, input loading, ANSI-aware line wrapping, search engine, follow mode, and screen rendering. Thin console app (`less`) does arg parsing via ShellKit, calls the library, and sets exit codes. The pager reads piped stdin or a file argument, displays content interactively with scrolling and search, and falls back to direct output for short content (quit-if-one-screen).

**Tech Stack:** .NET 10 / C# / xUnit. AOT-ready. No external dependencies beyond ShellKit.

```csharp
// src/Winix.Less/LessOptions.cs
namespace Winix.Less;

/// <summary>
/// Resolved pager options. Combines built-in defaults, <c>LESS</c> environment variable,
/// and command-line flags with correct precedence.
/// </summary>
public sealed record LessOptions
{
    /// <summary>Show line numbers in a left gutter (<c>-N</c>).</summary>
    public bool ShowLineNumbers { get; init; }

    /// <summary>Truncate long lines instead of wrapping (<c>-S</c>).</summary>
    public bool ChopLongLines { get; init; }

    /// <summary>Quit immediately if content fits on one screen (<c>-F</c>). Default: true.</summary>
    public bool QuitIfOneScreen { get; init; }

    /// <summary>Pass ANSI escape sequences through to the terminal (<c>-R</c>). Default: true.</summary>
    public bool RawAnsi { get; init; }

    /// <summary>Don't clear the screen on exit (<c>-X</c>). Default: true.</summary>
    public bool NoClearOnExit { get; init; }

    /// <summary>Case-insensitive search unless pattern contains uppercase (<c>-i</c>, smart case).</summary>
    public bool IgnoreCase { get; init; }

    /// <summary>Always case-insensitive search, even with uppercase in pattern (<c>-I</c>).</summary>
    public bool ForceIgnoreCase { get; init; }

    /// <summary>Start in follow mode (<c>+F</c>).</summary>
    public bool FollowOnStart { get; init; }

    /// <summary>Initial search pattern (<c>+/pattern</c>).</summary>
    public string? InitialSearch { get; init; }

    /// <summary>Start at end of file (<c>+G</c>).</summary>
    public bool StartAtEnd { get; init; }

    /// <summary>
    /// Resolves options from built-in defaults, <c>LESS</c> environment variable, and CLI flags.
    /// </summary>
    /// <remarks>
    /// <para>Precedence: CLI flags &gt; <c>LESS</c> env var &gt; built-in defaults.</para>
    /// <para>When <c>LESS</c> is unset or empty, built-in defaults apply (F, R, X on).
    /// When <c>LESS</c> is non-empty, it replaces defaults entirely — only the listed
    /// flags are enabled. CLI flags then override on top.</para>
    /// </remarks>
    /// <param name="cliFlags">Flags from the command line (e.g. "-N", "-S", "+F", "+/pattern").</param>
    /// <param name="lessEnvVar">Value of the <c>LESS</c> environment variable, or null if unset.</param>
    public static LessOptions Resolve(string[] cliFlags, string? lessEnvVar)
    {
        // Step 1: Determine base flags from defaults or env var
        bool flagN = false;
        bool flagS = false;
        bool flagF = false;
        bool flagR = false;
        bool flagX = false;
        bool flagI = false;
        bool flagBigI = false;

        if (string.IsNullOrEmpty(lessEnvVar))
        {
            // Built-in modern defaults
            flagF = true;
            flagR = true;
            flagX = true;
        }
        else
        {
            // Env var replaces defaults entirely
            foreach (char c in lessEnvVar)
            {
                switch (c)
                {
                    case 'N': flagN = true; break;
                    case 'S': flagS = true; break;
                    case 'F': flagF = true; break;
                    case 'R': flagR = true; break;
                    case 'X': flagX = true; break;
                    case 'i': flagI = true; break;
                    case 'I': flagBigI = true; break;
                    default: break; // Unknown flags silently ignored
                }
            }
        }

        // Step 2: CLI flags override on top
        bool followOnStart = false;
        bool startAtEnd = false;
        string? initialSearch = null;

        foreach (string flag in cliFlags)
        {
            switch (flag)
            {
                case "-N": flagN = true; break;
                case "-S": flagS = true; break;
                case "-F": flagF = true; break;
                case "-R": flagR = true; break;
                case "-X": flagX = true; break;
                case "-i": flagI = true; break;
                case "-I": flagBigI = true; break;
                case "+F": followOnStart = true; break;
                case "+G": startAtEnd = true; break;
                default:
                    if (flag.StartsWith("+/") && flag.Length > 2)
                    {
                        initialSearch = flag.Substring(2);
                    }
                    break;
            }
        }

        return new LessOptions
        {
            ShowLineNumbers = flagN,
            ChopLongLines = flagS,
            QuitIfOneScreen = flagF,
            RawAnsi = flagR,
            NoClearOnExit = flagX,
            IgnoreCase = flagI,
            ForceIgnoreCase = flagBigI,
            FollowOnStart = followOnStart,
            InitialSearch = initialSearch,
            StartAtEnd = startAtEnd,
        };
    }
}
```

```csharp
// src/Winix.Less/AnsiText.cs
using System.Text;
using System.Text.RegularExpressions;

namespace Winix.Less;

/// <summary>
/// Utilities for working with text that contains ANSI escape sequences.
/// All methods treat escape sequences as zero-width — they are preserved in output
/// but excluded from width calculations.
/// </summary>
public static class AnsiText
{
    private static readonly Regex AnsiEscapePattern = new Regex(
        @"\x1b\[[0-9;]*m", RegexOptions.Compiled);

    /// <summary>
    /// Strips all ANSI escape sequences from <paramref name="text"/>,
    /// returning only visible characters.
    /// </summary>
    public static string StripAnsi(string text)
    {
        return AnsiEscapePattern.Replace(text, "");
    }

    /// <summary>
    /// Returns the visible character count of <paramref name="text"/>,
    /// excluding ANSI escape sequences.
    /// </summary>
    public static int VisibleLength(string text)
    {
        return StripAnsi(text).Length;
    }

    /// <summary>
    /// Truncates <paramref name="text"/> so that its visible character count
    /// does not exceed <paramref name="maxWidth"/>. ANSI escape sequences
    /// within the visible range are preserved.
    /// </summary>
    public static string TruncateToWidth(string text, int maxWidth)
    {
        if (maxWidth <= 0)
        {
            return "";
        }

        var sb = new StringBuilder();
        int visibleCount = 0;
        int pos = 0;

        while (pos < text.Length && visibleCount < maxWidth)
        {
            if (text[pos] == '\x1b')
            {
                // Copy the entire ANSI escape sequence verbatim
                int escEnd = pos + 1;
                if (escEnd < text.Length && text[escEnd] == '[')
                {
                    escEnd++;
                    while (escEnd < text.Length && text[escEnd] != 'm')
                    {
                        escEnd++;
                    }
                    if (escEnd < text.Length)
                    {
                        escEnd++; // include the 'm'
                    }
                }
                sb.Append(text.Substring(pos, escEnd - pos));
                pos = escEnd;
            }
            else
            {
                sb.Append(text[pos]);
                visibleCount++;
                pos++;
            }
        }

        return sb.ToString();
    }

    /// <summary>
    /// Returns a substring of <paramref name="text"/> starting at the given
    /// visible character offset. ANSI escapes before the offset are skipped;
    /// escapes after the offset are preserved.
    /// Used for horizontal scrolling in chop mode (<c>-S</c>).
    /// </summary>
    public static string SubstringByVisibleOffset(string text, int visibleOffset)
    {
        int visible = 0;
        int pos = 0;

        // Skip past the offset
        while (pos < text.Length && visible < visibleOffset)
        {
            if (text[pos] == '\x1b')
            {
                int escEnd = pos + 1;
                if (escEnd < text.Length && text[escEnd] == '[')
                {
                    escEnd++;
                    while (escEnd < text.Length && text[escEnd] != 'm')
                    {
                        escEnd++;
                    }
                    if (escEnd < text.Length)
                    {
                        escEnd++;
                    }
                }
                pos = escEnd;
            }
            else
            {
                visible++;
                pos++;
            }
        }

        return text.Substring(pos);
    }
}
```

```csharp
// src/Winix.Less/LineWrapper.cs
using System.Text;

namespace Winix.Less;

/// <summary>
/// Line display logic: soft-wrapping, chopping (horizontal scroll), line number
/// formatting, and display row calculation.
/// </summary>
public static class LineWrapper
{
    /// <summary>
    /// Soft-wraps a single line into one or more display rows, breaking at
    /// <paramref name="width"/> visible characters. ANSI escapes are preserved
    /// and do not count toward width.
    /// </summary>
    public static IReadOnlyList<string> WrapLine(string line, int width)
    {
        if (width <= 0)
        {
            return new[] { line };
        }

        int visibleLen = AnsiText.VisibleLength(line);
        if (visibleLen <= width)
        {
            return new[] { line };
        }

        var rows = new List<string>();
        int offset = 0;

        while (offset < visibleLen)
        {
            int remaining = visibleLen - offset;
            int take = Math.Min(remaining, width);

            string segment = AnsiText.SubstringByVisibleOffset(line, offset);
            rows.Add(AnsiText.TruncateToWidth(segment, take));
            offset += take;
        }

        return rows;
    }

    /// <summary>
    /// Chops a line for horizontal-scroll mode (<c>-S</c>). Returns the visible
    /// portion starting at <paramref name="leftColumn"/>, truncated to <paramref name="width"/>.
    /// </summary>
    public static string ChopLine(string line, int width, int leftColumn)
    {
        string shifted = AnsiText.SubstringByVisibleOffset(line, leftColumn);
        return AnsiText.TruncateToWidth(shifted, width);
    }

    /// <summary>
    /// Formats a line number for the gutter. Returns a right-aligned number string
    /// padded to <paramref name="gutterWidth"/> characters, followed by a space.
    /// If <paramref name="lineNumber"/> is null, returns a blank gutter (for continuation rows).
    /// </summary>
    public static string FormatLineNumber(int? lineNumber, int gutterWidth)
    {
        if (lineNumber is null)
        {
            return new string(' ', gutterWidth + 1);
        }

        string num = lineNumber.Value.ToString();
        int padding = gutterWidth - num.Length;
        if (padding < 0)
        {
            padding = 0;
        }

        return new string(' ', padding) + num + " ";
    }

    /// <summary>
    /// Calculates the total number of display rows needed to render all lines
    /// with soft-wrapping at the given width.
    /// </summary>
    public static int CalculateDisplayRows(IReadOnlyList<string> lines, int width)
    {
        int total = 0;
        foreach (string line in lines)
        {
            int visibleLen = AnsiText.VisibleLength(line);
            if (visibleLen <= width || width <= 0)
            {
                total++;
            }
            else
            {
                total += (visibleLen + width - 1) / width;
            }
        }
        return total;
    }
}
```

```csharp
// src/Winix.Less/InputSource.cs
using System.Text;

namespace Winix.Less;

/// <summary>
/// Loads content from a file or stdin into a line buffer.
/// Supports polling for new content (for follow mode).
/// </summary>
public sealed class InputSource
{
    private readonly string? _filePath;
    private readonly List<string> _lines;
    private long _lastFileLength;

    /// <summary>Display name: file path or "(stdin)".</summary>
    public string Name { get; }

    /// <summary>All lines currently loaded.</summary>
    public IReadOnlyList<string> Lines => _lines;

    private InputSource(string name, List<string> lines, string? filePath, long lastFileLength)
    {
        Name = name;
        _lines = lines;
        _filePath = filePath;
        _lastFileLength = lastFileLength;
    }

    /// <summary>
    /// Creates an input source from a file. Reads all content into memory.
    /// </summary>
    /// <exception cref="FileNotFoundException">The file does not exist.</exception>
    public static InputSource FromFile(string filePath)
    {
        if (!File.Exists(filePath))
        {
            throw new FileNotFoundException($"File not found: {filePath}", filePath);
        }

        string content = File.ReadAllText(filePath, Encoding.UTF8);
        long length = new FileInfo(filePath).Length;
        var lines = SplitLines(content);
        return new InputSource(filePath, lines, filePath, length);
    }

    /// <summary>
    /// Creates an input source from stdin. Reads until EOF.
    /// </summary>
    public static InputSource FromStdin()
    {
        string content = Console.In.ReadToEnd();
        var lines = SplitLines(content);
        return new InputSource("(stdin)", lines, null, 0);
    }

    /// <summary>
    /// Creates an input source from a string (for testing).
    /// </summary>
    public static InputSource FromString(string content, string name)
    {
        var lines = SplitLines(content);
        return new InputSource(name, lines, null, 0);
    }

    /// <summary>
    /// Checks whether the source file has grown since last read.
    /// Returns true if new lines were appended. No-op for stdin sources.
    /// </summary>
    public bool PollForNewContent()
    {
        if (_filePath is null)
        {
            return false;
        }

        var info = new FileInfo(_filePath);
        if (!info.Exists || info.Length <= _lastFileLength)
        {
            return false;
        }

        // Read only the new bytes from where we left off
        using var fs = new FileStream(_filePath, FileMode.Open, FileAccess.Read, FileShare.ReadWrite);
        fs.Seek(_lastFileLength, SeekOrigin.Begin);
        using var reader = new StreamReader(fs, Encoding.UTF8);
        string newContent = reader.ReadToEnd();
        _lastFileLength = info.Length;

        if (newContent.Length == 0)
        {
            return false;
        }

        var newLines = SplitLines(newContent);

        // If the previous last line was empty (trailing newline), merge
        if (_lines.Count > 0 && _lines[_lines.Count - 1].Length == 0 && newLines.Count > 0)
        {
            _lines[_lines.Count - 1] = newLines[0];
            for (int i = 1; i < newLines.Count; i++)
            {
                _lines.Add(newLines[i]);
            }
        }
        else
        {
            foreach (string line in newLines)
            {
                _lines.Add(line);
            }
        }

        return true;
    }

    private static List<string> SplitLines(string content)
    {
        // Handle both \r\n and \n line endings
        var lines = new List<string>(content.Split(new[] { "\r\n", "\n" }, StringSplitOptions.None));

        // Remove trailing empty line from final newline (but keep empty files as one empty line)
        if (lines.Count > 1 && lines[lines.Count - 1].Length == 0)
        {
            lines.RemoveAt(lines.Count - 1);
        }

        return lines;
    }
}
```

```csharp
// src/Winix.Less/SearchEngine.cs
namespace Winix.Less;

/// <summary>
/// Forward and backward text search through a line buffer.
/// Searches operate on ANSI-stripped visible text.
/// </summary>
public sealed class SearchEngine
{
    /// <summary>The most recent search pattern, or null if no search has been performed.</summary>
    public string? CurrentPattern { get; private set; }

    /// <summary>
    /// When true, search is case-insensitive unless <see cref="SmartCase"/> is active
    /// and the pattern contains uppercase characters. Corresponds to <c>-i</c>.
    /// </summary>
    public bool IgnoreCase { get; set; }

    /// <summary>
    /// When true, search is case-insensitive only if the pattern is all lowercase.
    /// If the pattern contains uppercase, search is case-sensitive.
    /// Corresponds to <c>-i</c> (smart case). Takes precedence over <see cref="IgnoreCase"/>.
    /// </summary>
    public bool SmartCase { get; set; }

    /// <summary>
    /// Searches forward from <paramref name="startLine"/> for the first line
    /// containing <paramref name="pattern"/>. Wraps around to the beginning.
    /// Returns null if no match is found.
    /// </summary>
    public int? FindNext(IReadOnlyList<string> lines, string pattern, int startLine)
    {
        CurrentPattern = pattern;
        var comparison = ResolveComparison(pattern);

        // Search from startLine to end
        for (int i = startLine; i < lines.Count; i++)
        {
            if (LineMatches(lines[i], pattern, comparison))
            {
                return i;
            }
        }

        // Wrap around from beginning to startLine
        for (int i = 0; i < startLine; i++)
        {
            if (LineMatches(lines[i], pattern, comparison))
            {
                return i;
            }
        }

        return null;
    }

    /// <summary>
    /// Searches backward from <paramref name="startLine"/> for the first line
    /// containing <paramref name="pattern"/>. Wraps around to the end.
    /// Returns null if no match is found.
    /// </summary>
    public int? FindPrevious(IReadOnlyList<string> lines, string pattern, int startLine)
    {
        CurrentPattern = pattern;
        var comparison = ResolveComparison(pattern);

        // Search from startLine backward to 0
        for (int i = startLine; i >= 0; i--)
        {
            if (LineMatches(lines[i], pattern, comparison))
            {
                return i;
            }
        }

        // Wrap around from end to startLine
        for (int i = lines.Count - 1; i > startLine; i--)
        {
            if (LineMatches(lines[i], pattern, comparison))
            {
                return i;
            }
        }

        return null;
    }

    private StringComparison ResolveComparison(string pattern)
    {
        if (SmartCase)
        {
            // Smart case: case-insensitive only if pattern is all lowercase
            bool hasUpper = false;
            foreach (char c in pattern)
            {
                if (char.IsUpper(c))
                {
                    hasUpper = true;
                    break;
                }
            }
            return hasUpper ? StringComparison.Ordinal : StringComparison.OrdinalIgnoreCase;
        }

        if (IgnoreCase)
        {
            return StringComparison.OrdinalIgnoreCase;
        }

        return StringComparison.Ordinal;
    }

    private static bool LineMatches(string line, string pattern, StringComparison comparison)
    {
        string visible = AnsiText.StripAnsi(line);
        return visible.IndexOf(pattern, comparison) >= 0;
    }
}
```

```csharp
using System.Text;

namespace Winix.Less;

/// <summary>
/// Manages terminal rendering for the pager: draws visible lines,
/// status bar, and search prompt. Handles raw mode (cursor visibility).
/// </summary>
public sealed class Screen : IDisposable
{
    private readonly LessOptions _options;
    private bool _disposed;

    /// <summary>Terminal height in rows.</summary>
    public int Height { get; private set; }

    /// <summary>Terminal width in columns.</summary>
    public int Width { get; private set; }

    /// <summary>Number of rows available for content (Height minus status bar).</summary>
    public int ViewHeight => Height - 1;

    /// <summary>
    /// Initialises the screen and enters raw mode (hides cursor).
    /// </summary>
    public Screen(LessOptions options)
    {
        _options = options;
        RefreshDimensions();
        Console.Write("\x1b[?25l"); // hide cursor
    }

    /// <summary>
    /// Re-reads terminal dimensions. Call when the terminal may have been resized.
    /// </summary>
    public void RefreshDimensions()
    {
        try
        {
            Height = Console.WindowHeight;
            Width = Console.WindowWidth;
        }
        catch
        {
            Height = 24;
            Width = 80;
        }

        if (Height <= 0) { Height = 24; }
        if (Width <= 0) { Width = 80; }
    }

    /// <summary>
    /// Renders the visible portion of lines and the status bar.
    /// </summary>
    /// <param name="lines">All content lines.</param>
    /// <param name="topLine">Index of the first visible source line.</param>
    /// <param name="leftColumn">Horizontal scroll offset (for chop mode).</param>
    /// <param name="sourceName">File name or "(stdin)" for the status bar.</param>
    /// <param name="totalLines">Total number of source lines.</param>
    /// <param name="searchPattern">Current search term for the status bar, or null.</param>
    /// <param name="isFollowing">Whether follow mode is active.</param>
    /// <param name="isAtEnd">Whether the view is at the end of content.</param>
    public void Render(
        IReadOnlyList<string> lines,
        int topLine,
        int leftColumn,
        string sourceName,
        int totalLines,
        string? searchPattern,
        bool isFollowing,
        bool isAtEnd)
    {
        Console.SetCursorPosition(0, 0);

        int contentWidth = _options.ShowLineNumbers ? Width - 7 : Width;
        int linesDrawn = 0;

        for (int i = topLine; i < lines.Count && linesDrawn < ViewHeight; i++)
        {
            if (_options.ChopLongLines)
            {
                string gutter = _options.ShowLineNumbers
                    ? LineWrapper.FormatLineNumber(i + 1, 6)
                    : "";
                string chopped = LineWrapper.ChopLine(lines[i], contentWidth, leftColumn);
                WritePaddedLine(gutter + chopped);
                linesDrawn++;
            }
            else
            {
                var rows = LineWrapper.WrapLine(lines[i], contentWidth);
                for (int r = 0; r < rows.Count && linesDrawn < ViewHeight; r++)
                {
                    string gutter = _options.ShowLineNumbers
                        ? LineWrapper.FormatLineNumber(r == 0 ? i + 1 : null, 6)
                        : "";
                    WritePaddedLine(gutter + rows[r]);
                    linesDrawn++;
                }
            }
        }

        // Blank remaining lines
        while (linesDrawn < ViewHeight)
        {
            WritePaddedLine("~");
            linesDrawn++;
        }

        // Status bar
        DrawStatusBar(sourceName, topLine, totalLines, searchPattern, isFollowing, isAtEnd);
    }

    /// <summary>
    /// Reads a single keypress from the terminal.
    /// </summary>
    public ConsoleKeyInfo ReadKey()
    {
        return Console.ReadKey(intercept: true);
    }

    /// <summary>
    /// Displays a prompt character at the status bar and reads a line of text.
    /// Returns null if the user presses Escape.
    /// </summary>
    public string? ReadPrompt(char promptChar)
    {
        Console.Write("\x1b[?25h"); // show cursor
        try
        {
            Console.SetCursorPosition(0, Height - 1);
            Console.Write(new string(' ', Width));
            Console.SetCursorPosition(0, Height - 1);
            Console.Write(promptChar);

            var sb = new StringBuilder();
            while (true)
            {
                var key = Console.ReadKey(intercept: true);
                if (key.Key == ConsoleKey.Enter)
                {
                    break;
                }
                if (key.Key == ConsoleKey.Escape)
                {
                    return null;
                }
                if (key.Key == ConsoleKey.Backspace)
                {
                    if (sb.Length > 0)
                    {
                        sb.Remove(sb.Length - 1, 1);
                        int col = 1 + sb.Length;
                        Console.SetCursorPosition(col, Height - 1);
                        Console.Write(' ');
                        Console.SetCursorPosition(col, Height - 1);
                    }
                    continue;
                }
                if (key.KeyChar != '\0')
                {
                    sb.Append(key.KeyChar);
                    Console.Write(key.KeyChar);
                }
            }

            return sb.ToString();
        }
        finally
        {
            Console.Write("\x1b[?25l"); // hide cursor again
        }
    }

    /// <summary>Restores terminal state.</summary>
    public void Dispose()
    {
        if (!_disposed)
        {
            Console.Write("\x1b[?25h"); // show cursor
            if (!_options.NoClearOnExit)
            {
                Console.Clear();
            }
            _disposed = true;
        }
    }

    private void DrawStatusBar(
        string sourceName, int topLine, int totalLines,
        string? searchPattern, bool isFollowing, bool isAtEnd)
    {
        Console.SetCursorPosition(0, Height - 1);

        string status;
        if (isFollowing)
        {
            status = " Waiting for data... (press any key to stop)";
        }
        else if (isAtEnd)
        {
            status = " (END)";
        }
        else
        {
            int bottomLine = Math.Min(topLine + ViewHeight, totalLines);
            int pct = totalLines > 0 ? (bottomLine * 100 / totalLines) : 100;
            string searchInfo = searchPattern is not null ? $"  /{searchPattern}" : "";
            status = $" {sourceName}  Lines {topLine + 1}-{bottomLine}/{totalLines} ({pct}%){searchInfo}";
        }

        if (status.Length > Width)
        {
            status = status.Substring(0, Width);
        }

        string bar = status + new string(' ', Math.Max(0, Width - status.Length));
        Console.Write($"\x1b[7m{bar}\x1b[0m");
    }

    private void WritePaddedLine(string content)
    {
        int visible = AnsiText.VisibleLength(content);
        int padding = Width - visible;
        if (padding > 0)
        {
            Console.Write(content + new string(' ', padding));
        }
        else
        {
            Console.Write(content);
        }
        Console.WriteLine();
    }
}
```

```csharp
namespace Winix.Less;

/// <summary>
/// Watches the input source for new content and auto-scrolls to the bottom.
/// Activated by the <c>F</c> key or <c>+F</c> startup option.
/// </summary>
public static class FollowMode
{
    /// <summary>
    /// Enters follow mode, polling the input source for new content.
    /// Calls <paramref name="onNewContent"/> when new lines appear.
    /// Exits when <paramref name="checkForKeyPress"/> returns true.
    /// </summary>
    /// <param name="source">The input source to poll.</param>
    /// <param name="onNewContent">Callback invoked when new content is detected.</param>
    /// <param name="checkForKeyPress">Returns true if the user pressed a key to exit follow mode.</param>
    public static void Enter(
        InputSource source,
        Action onNewContent,
        Func<bool> checkForKeyPress)
    {
        while (true)
        {
            if (checkForKeyPress())
            {
                return;
            }

            if (source.PollForNewContent())
            {
                onNewContent();
            }

            Thread.Sleep(250);
        }
    }
}
```

```csharp
// src/Winix.Less/Pager.cs
namespace Winix.Less;

/// <summary>
/// Main pager event loop. Coordinates input source, screen rendering,
/// search engine, and follow mode into an interactive viewing session.
/// </summary>
public sealed class Pager
{
    private LessOptions _options;

    /// <summary>Creates a pager with the given options.</summary>
    public Pager(LessOptions options)
    {
        _options = options;
    }

    /// <summary>
    /// Runs the pager on the given input source.
    /// Returns 0 on success.
    /// </summary>
    public int Run(InputSource source)
    {
        // Quit-if-one-screen: if content fits, dump and exit
        if (_options.QuitIfOneScreen)
        {
            int termHeight;
            try { termHeight = Console.WindowHeight; }
            catch { termHeight = 24; }

            int availableRows = termHeight - 1;
            if (source.Lines.Count <= availableRows)
            {
                foreach (string line in source.Lines)
                {
                    Console.Out.WriteLine(line);
                }
                return 0;
            }
        }

        var search = new SearchEngine
        {
            IgnoreCase = _options.ForceIgnoreCase,
            SmartCase = _options.IgnoreCase && !_options.ForceIgnoreCase,
        };

        using var screen = new Screen(_options);

        int topLine = _options.StartAtEnd
            ? Math.Max(0, source.Lines.Count - screen.ViewHeight)
            : 0;
        int leftColumn = 0;
        bool isFollowing = false;

        // Handle initial search
        if (_options.InitialSearch is not null)
        {
            int? match = search.FindNext(source.Lines, _options.InitialSearch, 0);
            if (match.HasValue)
            {
                topLine = match.Value;
            }
        }

        // Handle follow on start
        if (_options.FollowOnStart)
        {
            topLine = Math.Max(0, source.Lines.Count - screen.ViewHeight);
            isFollowing = true;
        }

        bool redraw = true;
        while (true)
        {
            screen.RefreshDimensions();
            int maxTop = Math.Max(0, source.Lines.Count - screen.ViewHeight);

            if (isFollowing)
            {
                // In follow mode
                screen.Render(source.Lines, topLine, leftColumn, source.Name,
                    source.Lines.Count, search.CurrentPattern, true, true);

                FollowMode.Enter(
                    source,
                    onNewContent: () =>
                    {
                        topLine = Math.Max(0, source.Lines.Count - screen.ViewHeight);
                        screen.RefreshDimensions();
                        screen.Render(source.Lines, topLine, leftColumn, source.Name,
                            source.Lines.Count, search.CurrentPattern, true, true);
                    },
                    checkForKeyPress: () => Console.KeyAvailable);

                // User pressed a key — exit follow mode
                Console.ReadKey(intercept: true); // consume the key
                isFollowing = false;
                topLine = Math.Max(0, source.Lines.Count - screen.ViewHeight);
                redraw = true;
                continue;
            }

            if (redraw)
            {
                bool isAtEnd = topLine >= maxTop;
                screen.Render(source.Lines, topLine, leftColumn, source.Name,
                    source.Lines.Count, search.CurrentPattern, false, isAtEnd);
                redraw = false;
            }

            var key = screen.ReadKey();

            switch (key.Key)
            {
                case ConsoleKey.Q:
                    return 0;

                case ConsoleKey.J:
                case ConsoleKey.DownArrow:
                case ConsoleKey.Enter:
                    if (topLine < maxTop)
                    {
                        topLine++;
                        redraw = true;
                    }
                    break;

                case ConsoleKey.K:
                case ConsoleKey.UpArrow:
                    if (topLine > 0)
                    {
                        topLine--;
                        redraw = true;
                    }
                    break;

                case ConsoleKey.Spacebar:
                case ConsoleKey.PageDown:
                    topLine = Math.Min(topLine + screen.ViewHeight, maxTop);
                    redraw = true;
                    break;

                case ConsoleKey.PageUp:
                    topLine = Math.Max(0, topLine - screen.ViewHeight);
                    redraw = true;
                    break;

                case ConsoleKey.Home:
                    topLine = 0;
                    leftColumn = 0;
                    redraw = true;
                    break;

                case ConsoleKey.End:
                    topLine = maxTop;
                    redraw = true;
                    break;

                case ConsoleKey.RightArrow:
                    if (_options.ChopLongLines)
                    {
                        leftColumn += 8;
                        redraw = true;
                    }
                    break;

                case ConsoleKey.LeftArrow:
                    if (_options.ChopLongLines && leftColumn > 0)
                    {
                        leftColumn = Math.Max(0, leftColumn - 8);
                        redraw = true;
                    }
                    break;

                default:
                    redraw = HandleCharKey(key, screen, search, source, ref topLine, ref leftColumn, ref isFollowing);
                    break;
            }
        }
    }

    private bool HandleCharKey(
        ConsoleKeyInfo key,
        Screen screen,
        SearchEngine search,
        InputSource source,
        ref int topLine,
        ref int leftColumn,
        ref bool isFollowing)
    {
        switch (key.KeyChar)
        {
            case 'g':
                topLine = 0;
                leftColumn = 0;
                return true;

            case 'G':
                topLine = Math.Max(0, source.Lines.Count - screen.ViewHeight);
                return true;

            case 'd':
                topLine = Math.Min(topLine + screen.ViewHeight,
                    Math.Max(0, source.Lines.Count - screen.ViewHeight));
                return true;

            case 'u':
                topLine = Math.Max(0, topLine - screen.ViewHeight);
                return true;

            case '/':
            {
                string? pattern = screen.ReadPrompt('/');
                if (pattern is not null && pattern.Length > 0)
                {
                    int? match = search.FindNext(source.Lines, pattern, topLine + 1);
                    if (match.HasValue)
                    {
                        topLine = match.Value;
                    }
                }
                return true;
            }

            case '?':
            {
                string? pattern = screen.ReadPrompt('?');
                if (pattern is not null && pattern.Length > 0)
                {
                    int? match = search.FindPrevious(source.Lines, pattern, topLine - 1);
                    if (match.HasValue)
                    {
                        topLine = match.Value;
                    }
                }
                return true;
            }

            case 'n':
            {
                if (search.CurrentPattern is not null)
                {
                    int? match = search.FindNext(source.Lines, search.CurrentPattern, topLine + 1);
                    if (match.HasValue)
                    {
                        topLine = match.Value;
                    }
                }
                return true;
            }

            case 'N':
            {
                if (search.CurrentPattern is not null)
                {
                    int? match = search.FindPrevious(source.Lines, search.CurrentPattern, topLine - 1);
                    if (match.HasValue)
                    {
                        topLine = match.Value;
                    }
                }
                return true;
            }

            case 'F':
                isFollowing = true;
                topLine = Math.Max(0, source.Lines.Count - screen.ViewHeight);
                return true;

            case '-':
            {
                // Runtime toggle: read next char for the flag
                var nextKey = screen.ReadKey();
                if (nextKey.KeyChar == 'N')
                {
                    _options = _options with { ShowLineNumbers = !_options.ShowLineNumbers };
                }
                else if (nextKey.KeyChar == 'S')
                {
                    _options = _options with { ChopLongLines = !_options.ChopLongLines };
                    leftColumn = 0; // reset horizontal scroll when toggling chop
                }
                return true;
            }

            default:
                return false;
        }
    }
}
```

```csharp
using System.Reflection;
using Winix.Less;
using Yort.ShellKit;

namespace Less;

internal sealed class Program
{
    static int Main(string[] args)
    {
        ConsoleEnv.EnableAnsiIfNeeded();
        string version = GetVersion();

        var parser = new CommandLineParser("less", version)
            .Description("Display file contents one screen at a time with scrolling, search, and ANSI colour passthrough.")
            .StandardFlags()
            .Flag("-N", "Show line numbers")
            .Flag("-S", "Chop (truncate) long lines instead of wrapping")
            .Flag("-F", "Quit if content fits on one screen (default: on)")
            .Flag("-R", "Raw ANSI passthrough (default: on)")
            .Flag("-X", "Don't clear screen on exit (default: on)")
            .Flag("-i", "Case-insensitive search (smart case: sensitive if pattern has uppercase)")
            .Flag("-I", "Force case-insensitive search")
            .Positional("[file]")
            .Platform("cross-platform",
                new[] { "less", "more" },
                "Windows has no native pager — more.com is forward-only and destroys ANSI",
                "Native ANSI passthrough, search, follow mode, modern defaults")
            .StdinDescription("Content to page (when no file argument)")
            .StdoutDescription("Paged content (with -F for short content, or when piped)")
            .StderrDescription("Errors")
            .Example("git diff | less", "Page coloured diff output")
            .Example("less logfile.txt", "View a file with scrolling and search")
            .Example("less -N logfile.txt", "View with line numbers")
            .Example("less -S wide.csv", "View wide file with horizontal scrolling")
            .Example("less +F logfile.txt", "Follow file for new content (like tail -f)")
            .Example("less +/error logfile.txt", "Open file and jump to first 'error'")
            .ExitCodes(
                (0, "Success"),
                (1, "Error (file not found, read error)"),
                (2, "Usage error"));

        var result = parser.Parse(args);
        if (result.IsHandled) { return result.ExitCode; }
        if (result.HasErrors) { return result.WriteErrors(Console.Error); }

        // Collect CLI flags for LessOptions (the flags that are less-specific, not ShellKit standard)
        var lessFlags = new List<string>();
        if (result.Has("-N")) { lessFlags.Add("-N"); }
        if (result.Has("-S")) { lessFlags.Add("-S"); }
        if (result.Has("-F")) { lessFlags.Add("-F"); }
        if (result.Has("-R")) { lessFlags.Add("-R"); }
        if (result.Has("-X")) { lessFlags.Add("-X"); }
        if (result.Has("-i")) { lessFlags.Add("-i"); }
        if (result.Has("-I")) { lessFlags.Add("-I"); }

        // Check positionals for +F, +G, +/pattern (these look like positionals to the parser)
        string? filePath = null;
        foreach (string pos in result.Positionals)
        {
            if (pos == "+F")
            {
                lessFlags.Add("+F");
            }
            else if (pos == "+G")
            {
                lessFlags.Add("+G");
            }
            else if (pos.StartsWith("+/") && pos.Length > 2)
            {
                lessFlags.Add(pos);
            }
            else
            {
                filePath = pos;
            }
        }

        // Resolve options
        string? lessEnv = Environment.GetEnvironmentVariable("LESS");
        var options = LessOptions.Resolve(lessFlags.ToArray(), lessEnv);

        // Load input
        InputSource source;
        try
        {
            if (filePath is not null)
            {
                source = InputSource.FromFile(filePath);
            }
            else if (Console.IsInputRedirected)
            {
                source = InputSource.FromStdin();
            }
            else
            {
                Console.Error.WriteLine("less: missing filename (use \"less --help\" for help)");
                return 2;
            }
        }
        catch (FileNotFoundException ex)
        {
            Console.Error.WriteLine($"less: {ex.Message}");
            return 1;
        }

        // Run the pager
        var pager = new Pager(options);
        return pager.Run(source);
    }

    private static string GetVersion()
    {
        return typeof(Pager).Assembly
            .GetCustomAttribute<AssemblyInformationalVersionAttribute>()
            ?.InformationalVersion ?? "0.0.0";
    }
}
```

## Task 12: ADR

**Files:**
- Create: `docs/plans/2026-04-11-less-adr.md`

- [ ] **Step 1: Write the Architecture Decision Record**

Create `docs/plans/2026-04-11-less-adr.md` following the project's ADR format (see `docs/plans/2026-03-31-man-adr.md` for the pattern). Cover:

1. **Windows-primary pager** — Context: Windows has no native pager. Decision: Build a native less. Rationale: fills a real gap. Trade-off: limited value on Linux/macOS where GNU less exists.
2. **Modern defaults (FRX)** — Context: GNU less defaults are 40 years old. Decision: Default to quit-if-one-screen, raw ANSI, no clear on exit. Rationale: matches git's LESS=FRX convention.
3. **LESS env var as override, not additive** — Context: Need to support existing LESS env var setups. Decision: Non-empty LESS replaces defaults entirely; empty/unset uses modern defaults. Rationale: declarative, compatible with GNU less semantics.
4. **In-memory content loading** — Context: Need to decide between streaming and buffered. Decision: Load entirely into memory for v1. Rationale: simple, 100MB files are still fine, streaming can be added later.
5. **Reusable AnsiText utilities** — Context: ANSI stripping/width logic duplicated across man, peep, less. Decision: Create AnsiText in Winix.Less for now. Trade-off: deferred moving to ShellKit (would be a cross-cutting refactor).

Deferred decisions table: streaming for large files, ShellKit ANSI utilities extraction, regex search, search highlighting.
