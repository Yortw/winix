# Man Page Viewer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a cross-platform man page viewer that parses groff man pages and renders them to the terminal with ANSI colour and OSC 8 hyperlinks.

**Architecture:** Three-stage pipeline (lexer → macro expander → renderer) with an intermediate representation that decouples parsing from output. Page discovery searches bundled pages, `MANPATH`, and auto-detected locations. A pager chain falls back from external pagers to a built-in minimal pager.

**Tech Stack:** .NET 10, AOT-compiled, Yort.ShellKit for CLI parsing and terminal detection, xUnit for tests.

**Spec:** `docs/plans/2026-03-31-man-design.md`

// src/Winix.Man/GroffLexer.cs
namespace Winix.Man;

/// <summary>
/// Tokenises raw groff source into a stream of <see cref="GroffToken"/> instances.
/// Each input line becomes one token. Does not interpret macros — only identifies
/// requests, text lines, and comments.
/// </summary>
public sealed class GroffLexer
{
    /// <summary>
    /// Tokenises groff source text into a sequence of tokens, one per input line.
    /// </summary>
    public IEnumerable<GroffToken> Tokenise(string source)
    {
        using var reader = new StringReader(source);
        string? line;
        while ((line = reader.ReadLine()) is not null)
        {
            yield return TokeniseLine(line);
        }
    }

    private static GroffToken TokeniseLine(string line)
    {
        if (line.Length == 0)
        {
            return new TextLineToken("");
        }

        // Request lines start with '.' or '\''
        if (line[0] == '.' || line[0] == '\'')
        {
            return ParseRequest(line);
        }

        return new TextLineToken(line);
    }

    private static GroffToken ParseRequest(string line)
    {
        // Comment: .\" or .\"
        if (line.Length >= 3 && line[0] == '.' && line[1] == '\\' && line[2] == '"')
        {
            return new CommentToken(line.Length > 3 ? line[4..] : "");
        }

        // Skip the control character (. or ')
        int pos = 1;

        // Skip leading whitespace after control character
        while (pos < line.Length && line[pos] == ' ')
        {
            pos++;
        }

        // Read macro name (letters and digits, typically 1-2 chars)
        int nameStart = pos;
        while (pos < line.Length && line[pos] != ' ' && line[pos] != '\t')
        {
            pos++;
        }

        if (pos == nameStart)
        {
            // Bare '.' or '\'' with no macro name — treat as empty text
            return new TextLineToken(line);
        }

        string macroName = line[nameStart..pos];

        // Skip whitespace between macro name and arguments
        while (pos < line.Length && (line[pos] == ' ' || line[pos] == '\t'))
        {
            pos++;
        }

        string arguments = pos < line.Length ? line[pos..] : "";

        return new RequestToken(macroName, arguments);
    }
}
```

// src/Winix.Man/ManMacroExpander.cs
using System.Text;

namespace Winix.Man;

/// <summary>
/// Expands groff <c>man</c> macro tokens into an intermediate representation
/// of <see cref="DocumentBlock"/> instances. Handles the standard man macro set
/// (.TH, .SH, .SS, .PP, .TP, .IP, .RS, .RE, .B, .I, .BR, .BI, .nf, .fi, .sp, etc.).
/// </summary>
public sealed class ManMacroExpander
{
    /// <summary>
    /// Expands a stream of groff tokens into a sequence of document blocks.
    /// </summary>
    public IReadOnlyList<DocumentBlock> Expand(IEnumerable<GroffToken> tokens)
    {
        var blocks = new List<DocumentBlock>();
        var currentSpans = new List<StyledSpan>();
        int indentLevel = 0;
        int? pendingIndent = null; // When set, next flush produces IndentedParagraph
        bool inNoFill = false;
        StringBuilder? noFillBuilder = null;
        bool expectingTag = false;
        List<StyledSpan>? tagSpans = null;

        foreach (GroffToken token in tokens)
        {
            if (token is CommentToken)
            {
                continue;
            }

            if (inNoFill)
            {
                if (token is RequestToken req && req.MacroName == "fi")
                {
                    inNoFill = false;
                    if (noFillBuilder is not null && noFillBuilder.Length > 0)
                    {
                        blocks.Add(new PreformattedBlock(noFillBuilder.ToString()));
                    }
                    noFillBuilder = null;
                    continue;
                }

                noFillBuilder ??= new StringBuilder();
                if (noFillBuilder.Length > 0)
                {
                    noFillBuilder.AppendLine();
                }

                string lineText = token is TextLineToken txt ? txt.Text : token.ToString() ?? "";
                // Process inline escapes for the raw text
                noFillBuilder.Append(ResolveInlineEscapesAsPlainText(lineText));
                continue;
            }

            if (expectingTag && token is TextLineToken tagText)
            {
                tagSpans = ParseInlineContent(tagText.Text);
                expectingTag = false;
                continue;
            }

            if (tagSpans is not null)
            {
                // We have a tag — collect body lines until next request
                if (token is TextLineToken bodyText)
                {
                    currentSpans.AddRange(ParseInlineContent(bodyText.Text));
                    continue;
                }

                // Request encountered — flush the tagged paragraph
                FlushTaggedParagraph(blocks, tagSpans, currentSpans);
                tagSpans = null;
                currentSpans.Clear();
                // Fall through to process the request
            }

            if (token is RequestToken request)
            {
                switch (request.MacroName)
                {
                    case "TH":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        blocks.Add(ParseTitleHeader(request.Arguments));
                        break;

                    case "SH":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        blocks.Add(new SectionHeading(Unquote(request.Arguments)));
                        break;

                    case "SS":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        blocks.Add(new SubsectionHeading(Unquote(request.Arguments)));
                        break;

                    case "PP":
                    case "P":
                    case "LP":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        break;

                    case "TP":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        expectingTag = true;
                        break;

                    case "IP":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        // .IP [marker [indent]] — following text becomes IndentedParagraph
                        {
                            var ipArgs = SplitArguments(request.Arguments);
                            pendingIndent = indentLevel + 2;
                            if (ipArgs.Count >= 2 && int.TryParse(ipArgs[1], out int parsedIndent))
                            {
                                pendingIndent = indentLevel + parsedIndent;
                            }
                            if (ipArgs.Count > 0 && ipArgs[0].Length > 0)
                            {
                                string marker = ResolveInlineEscapesAsPlainText(Unquote(ipArgs[0]));
                                currentSpans.Add(new StyledSpan(marker + " ", FontStyle.Roman));
                            }
                        }
                        break;

                    case "RS":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        indentLevel++;
                        break;

                    case "RE":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        if (indentLevel > 0)
                        {
                            indentLevel--;
                        }
                        break;

                    case "B":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        if (request.Arguments.Length > 0)
                        {
                            blocks.Add(new Paragraph(new[] { new StyledSpan(Unquote(request.Arguments), FontStyle.Bold) }));
                        }
                        break;

                    case "I":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        if (request.Arguments.Length > 0)
                        {
                            blocks.Add(new Paragraph(new[] { new StyledSpan(Unquote(request.Arguments), FontStyle.Italic) }));
                        }
                        break;

                    case "BI":
                    case "BR":
                    case "IB":
                    case "IR":
                    case "RB":
                    case "RI":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        blocks.Add(new Paragraph(ParseAlternatingFonts(request.MacroName, request.Arguments)));
                        break;

                    case "nf":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        inNoFill = true;
                        noFillBuilder = new StringBuilder();
                        break;

                    case "fi":
                        // fi without nf — ignore
                        break;

                    case "sp":
                        FlushParagraph(blocks, currentSpans, ref pendingIndent);
                        blocks.Add(new VerticalSpace(1));
                        break;

                    default:
                        // Unknown macro — treat arguments as text
                        if (request.Arguments.Length > 0)
                        {
                            currentSpans.AddRange(ParseInlineContent(request.Arguments));
                        }
                        break;
                }
            }
            else if (token is TextLineToken textLine)
            {
                if (textLine.Text.Length == 0)
                {
                    // Empty line acts like .PP
                    FlushParagraph(blocks, currentSpans, ref pendingIndent);
                }
                else
                {
                    // Check if this is an .IP body line
                    if (blocks.Count > 0 && blocks[^1] is IndentedParagraph)
                    {
                        // Append to existing IP
                        currentSpans.AddRange(ParseInlineContent(textLine.Text));
                    }
                    else
                    {
                        if (currentSpans.Count > 0)
                        {
                            // Add space between consecutive text lines
                            currentSpans.Add(new StyledSpan(" ", FontStyle.Roman));
                        }
                        currentSpans.AddRange(ParseInlineContent(textLine.Text));
                    }
                }
            }
        }

        // Flush any remaining state
        if (tagSpans is not null)
        {
            FlushTaggedParagraph(blocks, tagSpans, currentSpans);
        }
        else
        {
            FlushParagraph(blocks, currentSpans, ref pendingIndent);
        }

        if (inNoFill && noFillBuilder is not null && noFillBuilder.Length > 0)
        {
            blocks.Add(new PreformattedBlock(noFillBuilder.ToString()));
        }

        return blocks;
    }

    private static void FlushParagraph(List<DocumentBlock> blocks, List<StyledSpan> spans,
        ref int? pendingIndent)
    {
        if (spans.Count == 0)
        {
            pendingIndent = null;
            return;
        }

        if (pendingIndent.HasValue)
        {
            blocks.Add(new IndentedParagraph(spans.ToArray(), pendingIndent.Value));
            pendingIndent = null;
        }
        else
        {
            blocks.Add(new Paragraph(spans.ToArray()));
        }
        spans.Clear();
    }

    private static void FlushTaggedParagraph(
        List<DocumentBlock> blocks, List<StyledSpan> tag, List<StyledSpan> body)
    {
        blocks.Add(new TaggedParagraph(tag.ToArray(), body.ToArray()));
        body.Clear();
    }

    /// <summary>
    /// Parses inline text content, resolving \fB, \fI, \fR, \fP, \-, \\, \(xx escapes.
    /// </summary>
    internal static List<StyledSpan> ParseInlineContent(string text)
    {
        var spans = new List<StyledSpan>();
        var current = new StringBuilder();
        var currentStyle = FontStyle.Roman;
        var previousStyle = FontStyle.Roman;

        for (int i = 0; i < text.Length; i++)
        {
            if (text[i] == '\\' && i + 1 < text.Length)
            {
                char next = text[i + 1];

                if (next == 'f' && i + 2 < text.Length)
                {
                    // Font change: \fB, \fI, \fR, \fP
                    if (current.Length > 0)
                    {
                        spans.Add(new StyledSpan(current.ToString(), currentStyle));
                        current.Clear();
                    }

                    char fontChar = text[i + 2];
                    previousStyle = currentStyle;
                    currentStyle = fontChar switch
                    {
                        'B' => FontStyle.Bold,
                        'I' => FontStyle.Italic,
                        'R' => FontStyle.Roman,
                        'P' => previousStyle,
                        _ => FontStyle.Roman
                    };
                    i += 2;
                    continue;
                }

                if (next == '-')
                {
                    current.Append('-');
                    i++;
                    continue;
                }

                if (next == '\\')
                {
                    current.Append('\\');
                    i++;
                    continue;
                }

                if (next == 'e')
                {
                    current.Append('\\');
                    i++;
                    continue;
                }

                if (next == '(' && i + 3 < text.Length)
                {
                    // Two-character special: \(em, \(en, \(bu, etc.
                    string special = text.Substring(i + 2, 2);
                    current.Append(ResolveSpecialChar(special));
                    i += 3;
                    continue;
                }

                if (next == '&')
                {
                    // Zero-width space — skip
                    i++;
                    continue;
                }

                if (next == '~')
                {
                    // Non-breaking space
                    current.Append('\u00A0');
                    i++;
                    continue;
                }

                // Unknown escape — pass through
                current.Append(text[i]);
                continue;
            }

            current.Append(text[i]);
        }

        if (current.Length > 0)
        {
            spans.Add(new StyledSpan(current.ToString(), currentStyle));
        }

        return spans;
    }

    private static string ResolveSpecialChar(string code)
    {
        return code switch
        {
            "em" => "\u2014", // em-dash
            "en" => "\u2013", // en-dash
            "bu" => "\u2022", // bullet
            "lq" => "\u201C", // left double quote
            "rq" => "\u201D", // right double quote
            "aq" => "'",      // apostrophe quote
            _ => code
        };
    }

    private static string ResolveInlineEscapesAsPlainText(string text)
    {
        var spans = ParseInlineContent(text);
        return string.Join("", spans.Select(s => s.Text));
    }

    /// <summary>
    /// Parses .TH arguments: NAME SECTION [DATE [SOURCE [MANUAL]]].
    /// Arguments may be quoted.
    /// </summary>
    private static TitleBlock ParseTitleHeader(string args)
    {
        var parts = SplitArguments(args);
        string name = parts.Count > 0 ? Unquote(parts[0]) : "";
        string section = parts.Count > 1 ? Unquote(parts[1]) : "";
        string date = parts.Count > 2 ? Unquote(parts[2]) : "";
        string source = parts.Count > 3 ? Unquote(parts[3]) : "";
        string manual = parts.Count > 4 ? Unquote(parts[4]) : "";
        return new TitleBlock(name, section, date, source, manual);
    }

    /// <summary>
    /// Parses alternating-font macros (.BR, .BI, .IR, etc.).
    /// The macro name encodes which two fonts alternate: first char → odd args,
    /// second char → even args.
    /// </summary>
    private static List<StyledSpan> ParseAlternatingFonts(string macroName, string args)
    {
        FontStyle first = CharToFontStyle(macroName[0]);
        FontStyle second = CharToFontStyle(macroName[1]);

        var parts = SplitArguments(args);
        var spans = new List<StyledSpan>();

        for (int i = 0; i < parts.Count; i++)
        {
            FontStyle style = (i % 2 == 0) ? first : second;
            spans.Add(new StyledSpan(Unquote(parts[i]), style));
        }

        return spans;
    }

    private static FontStyle CharToFontStyle(char c)
    {
        return c switch
        {
            'B' => FontStyle.Bold,
            'I' => FontStyle.Italic,
            _ => FontStyle.Roman
        };
    }

    /// <summary>
    /// Splits arguments respecting double-quoted strings.
    /// "March 2026" is one argument, not two.
    /// </summary>
    internal static List<string> SplitArguments(string args)
    {
        var parts = new List<string>();
        var current = new StringBuilder();
        bool inQuotes = false;

        for (int i = 0; i < args.Length; i++)
        {
            char c = args[i];

            if (c == '"')
            {
                inQuotes = !inQuotes;
                current.Append(c);
                continue;
            }

            if ((c == ' ' || c == '\t') && !inQuotes)
            {
                if (current.Length > 0)
                {
                    parts.Add(current.ToString());
                    current.Clear();
                }
                continue;
            }

            current.Append(c);
        }

        if (current.Length > 0)
        {
            parts.Add(current.ToString());
        }

        return parts;
    }

    /// <summary>
    /// Removes surrounding double quotes from a string.
    /// </summary>
    internal static string Unquote(string text)
    {
        if (text.Length >= 2 && text[0] == '"' && text[^1] == '"')
        {
            return text[1..^1];
        }
        return text;
    }

}

// src/Winix.Man/TerminalRenderer.cs
using System.Text;

namespace Winix.Man;

/// <summary>
/// Renders <see cref="DocumentBlock"/> instances to ANSI-formatted terminal text.
/// Respects width limits, colour capability, and hyperlink support.
/// </summary>
public sealed class TerminalRenderer
{
    private const int DefaultIndent = 7;
    private const int TaggedBodyIndent = 15;
    private const int MaxWidth = 80;

    private readonly RendererOptions _options;
    private readonly int _width;

    /// <summary>
    /// Creates a renderer with the specified options.
    /// </summary>
    public TerminalRenderer(RendererOptions options)
    {
        _options = options;
        _width = options.WidthOverride ?? MaxWidth;
    }

    /// <summary>
    /// Renders document blocks to a string containing ANSI-formatted text.
    /// </summary>
    public string Render(IReadOnlyList<DocumentBlock> blocks)
    {
        if (blocks.Count == 0)
        {
            return "";
        }

        var sb = new StringBuilder();

        for (int i = 0; i < blocks.Count; i++)
        {
            RenderBlock(sb, blocks[i]);
        }

        return sb.ToString();
    }

    private void RenderBlock(StringBuilder sb, DocumentBlock block)
    {
        switch (block)
        {
            case TitleBlock title:
                RenderTitle(sb, title);
                break;
            case SectionHeading heading:
                RenderSectionHeading(sb, heading);
                break;
            case SubsectionHeading sub:
                RenderSubsectionHeading(sb, sub);
                break;
            case Paragraph para:
                RenderParagraph(sb, para, DefaultIndent);
                break;
            case TaggedParagraph tagged:
                RenderTaggedParagraph(sb, tagged);
                break;
            case IndentedParagraph indented:
                RenderParagraph(sb, new Paragraph(indented.Content), DefaultIndent + indented.Indent);
                break;
            case PreformattedBlock pre:
                RenderPreformatted(sb, pre);
                break;
            case VerticalSpace vs:
                for (int i = 0; i < vs.Lines; i++)
                {
                    sb.AppendLine();
                }
                break;
        }
    }

    private void RenderTitle(StringBuilder sb, TitleBlock title)
    {
        string left = $"{title.Name}({title.Section})";
        string centre = title.Manual.Length > 0 ? title.Manual : "";
        string right = left;

        // Distribute across the width
        int padding = _width - left.Length - centre.Length - right.Length;
        if (padding < 2)
        {
            // Not enough room — just print left
            sb.Append(Bold(left));
            sb.AppendLine();
        }
        else
        {
            int leftPad = padding / 2;
            int rightPad = padding - leftPad;
            sb.Append(Bold(left));
            sb.Append(new string(' ', leftPad));
            sb.Append(centre);
            sb.Append(new string(' ', rightPad));
            sb.Append(Bold(right));
            sb.AppendLine();
        }
        sb.AppendLine();
    }

    private void RenderSectionHeading(StringBuilder sb, SectionHeading heading)
    {
        sb.AppendLine();
        if (_options.Color)
        {
            sb.Append("\x1b[1m\x1b[36m");
            sb.Append(heading.Text);
            sb.Append("\x1b[0m");
        }
        else
        {
            sb.Append(Bold(heading.Text));
        }
        sb.AppendLine();
    }

    private void RenderSubsectionHeading(StringBuilder sb, SubsectionHeading heading)
    {
        sb.AppendLine();
        sb.Append(new string(' ', 3));
        sb.Append(Bold(heading.Text));
        sb.AppendLine();
    }

    private void RenderParagraph(StringBuilder sb, Paragraph para, int indent)
    {
        string plainText = RenderSpansToAnsi(para.Content);
        WrapAndAppend(sb, plainText, para.Content, indent);
    }

    private void RenderTaggedParagraph(StringBuilder sb, TaggedParagraph tagged)
    {
        // Render tag at DefaultIndent
        sb.Append(new string(' ', DefaultIndent));
        string tagText = RenderSpansToAnsi(tagged.Tag);
        sb.Append(tagText);

        // Calculate visible tag length (without ANSI escapes)
        int visibleTagLen = StripAnsi(tagText).Length;

        if (DefaultIndent + visibleTagLen >= TaggedBodyIndent - 1)
        {
            // Tag is too long — body goes on next line
            sb.AppendLine();
            sb.Append(new string(' ', TaggedBodyIndent));
        }
        else
        {
            // Body continues on same line
            int gap = TaggedBodyIndent - DefaultIndent - visibleTagLen;
            sb.Append(new string(' ', gap));
        }

        // Render body
        string bodyAnsi = RenderSpansToAnsi(tagged.Body);
        WrapAndAppend(sb, bodyAnsi, tagged.Body, TaggedBodyIndent, firstLineAlreadyIndented: true);
    }

    private void RenderPreformatted(StringBuilder sb, PreformattedBlock pre)
    {
        string indent = new string(' ', DefaultIndent);
        foreach (string line in pre.Text.Split('\n'))
        {
            sb.Append(indent);
            sb.AppendLine(line);
        }
    }

    /// <summary>
    /// Word-wraps styled content to the configured width, emitting ANSI codes inline.
    /// </summary>
    private void WrapAndAppend(StringBuilder sb, string ansiText, IReadOnlyList<StyledSpan> spans,
        int indent, bool firstLineAlreadyIndented = false)
    {
        // Get plain text for wrapping calculations
        string plain = string.Join("", spans.Select(s => s.Text));
        int availableWidth = _width - indent;

        if (availableWidth <= 0)
        {
            availableWidth = _width;
        }

        // Simple word-wrap on the plain text
        var lines = WordWrap(plain, availableWidth);

        for (int i = 0; i < lines.Count; i++)
        {
            if (i == 0 && firstLineAlreadyIndented)
            {
                sb.AppendLine(RenderSubstringAsAnsi(spans, lines[i]));
            }
            else
            {
                sb.Append(new string(' ', indent));
                sb.AppendLine(RenderSubstringAsAnsi(spans, lines[i]));
            }
        }
    }

    /// <summary>
    /// Renders a substring of the original text with the correct ANSI styles
    /// by mapping character positions back to spans.
    /// </summary>
    private string RenderSubstringAsAnsi(IReadOnlyList<StyledSpan> spans, string lineText)
    {
        // For simplicity, re-render the spans and extract the portion matching lineText.
        // This works because the line text is a substring of the concatenated span texts.
        var sb = new StringBuilder();
        int pos = 0;
        int linePos = 0;
        bool started = false;

        foreach (StyledSpan span in spans)
        {
            for (int i = 0; i < span.Text.Length && linePos < lineText.Length; i++)
            {
                if (!started)
                {
                    if (pos >= 0 && pos < int.MaxValue && span.Text[i] == lineText[linePos])
                    {
                        started = true;
                        sb.Append(StyleStart(span.Style));
                        sb.Append(span.Text[i]);
                        linePos++;
                    }
                }
                else if (span.Text[i] == lineText[linePos])
                {
                    sb.Append(span.Text[i]);
                    linePos++;
                }
                pos++;
            }

            if (started && sb.Length > 0)
            {
                sb.Append(StyleEnd(span.Style));
            }
        }

        // Fallback: if span mapping failed, just return the line with no styling
        if (linePos < lineText.Length)
        {
            return lineText;
        }

        return sb.ToString();
    }

    private string RenderSpansToAnsi(IReadOnlyList<StyledSpan> spans)
    {
        var sb = new StringBuilder();
        foreach (StyledSpan span in spans)
        {
            sb.Append(StyleStart(span.Style));
            sb.Append(span.Text);
            sb.Append(StyleEnd(span.Style));
        }
        return sb.ToString();
    }

    private string StyleStart(FontStyle style)
    {
        return style switch
        {
            FontStyle.Bold => "\x1b[1m",
            FontStyle.Italic => "\x1b[4m",
            FontStyle.Bold | FontStyle.Italic => "\x1b[1m\x1b[4m",
            _ => ""
        };
    }

    private string StyleEnd(FontStyle style)
    {
        if (style == FontStyle.Roman)
        {
            return "";
        }
        return "\x1b[0m";
    }

    private string Bold(string text)
    {
        return $"\x1b[1m{text}\x1b[0m";
    }

    /// <summary>
    /// Simple word-wrap: splits text into lines that fit within the given width.
    /// </summary>
    internal static List<string> WordWrap(string text, int width)
    {
        if (width <= 0 || text.Length == 0)
        {
            return new List<string> { text };
        }

        var lines = new List<string>();
        var currentLine = new StringBuilder();
        string[] words = text.Split(' ');

        foreach (string word in words)
        {
            if (word.Length == 0)
            {
                continue;
            }

            if (currentLine.Length == 0)
            {
                currentLine.Append(word);
            }
            else if (currentLine.Length + 1 + word.Length <= width)
            {
                currentLine.Append(' ');
                currentLine.Append(word);
            }
            else
            {
                lines.Add(currentLine.ToString());
                currentLine.Clear();
                currentLine.Append(word);
            }
        }

        if (currentLine.Length > 0)
        {
            lines.Add(currentLine.ToString());
        }

        return lines;
    }

    private static string StripAnsi(string text)
    {
        var sb = new StringBuilder();
        bool inEscape = false;

        for (int i = 0; i < text.Length; i++)
        {
            if (text[i] == '\x1b')
            {
                inEscape = true;
                continue;
            }

            if (inEscape)
            {
                if (text[i] == 'm')
                {
                    inEscape = false;
                }
                continue;
            }

            sb.Append(text[i]);
        }

        return sb.ToString();
    }
}

using System.IO.Compression;
using System.Text;

namespace Winix.Man;

/// <summary>
/// Reads man page files from disk. Transparently decompresses gzip-compressed
/// files (detected by .gz extension).
/// </summary>
public static class ManPageFileReader
{
    /// <summary>
    /// Reads the content of a man page file as a string.
    /// Files ending in .gz are decompressed automatically.
    /// </summary>
    /// <exception cref="FileNotFoundException">The file does not exist.</exception>
    public static string Read(string filePath)
    {
        if (!File.Exists(filePath))
        {
            throw new FileNotFoundException($"Man page not found: {filePath}", filePath);
        }

        if (filePath.EndsWith(".gz", StringComparison.OrdinalIgnoreCase))
        {
            using var fs = File.OpenRead(filePath);
            using var gz = new GZipStream(fs, CompressionMode.Decompress);
            using var reader = new StreamReader(gz, Encoding.UTF8);
            return reader.ReadToEnd();
        }

        return File.ReadAllText(filePath, Encoding.UTF8);
    }
}

using System.Runtime.InteropServices;

namespace Winix.Man;

/// <summary>
/// Auto-detection of well-known man page locations on each platform.
/// </summary>
public static class WellKnownPaths
{
    /// <summary>
    /// Returns auto-detected man page directories for the current platform.
    /// Only returns paths that actually exist on disk.
    /// </summary>
    public static List<string> Detect()
    {
        var paths = new List<string>();

        if (RuntimeInformation.IsOSPlatform(OSPlatform.Windows))
        {
            DetectWindows(paths);
        }
        else if (RuntimeInformation.IsOSPlatform(OSPlatform.OSX))
        {
            DetectMacOS(paths);
        }
        else
        {
            DetectLinux(paths);
        }

        return paths;
    }

    private static void DetectWindows(List<string> paths)
    {
        // Git for Windows
        string? gitPath = FindGitForWindowsInstall();
        if (gitPath is not null)
        {
            string gitManPath = Path.Combine(gitPath, "usr", "share", "man");
            if (Directory.Exists(gitManPath))
            {
                paths.Add(gitManPath);
            }
        }

        // MSYS2
        string? msys2Root = Environment.GetEnvironmentVariable("MSYS2_ROOT");
        if (msys2Root is null)
        {
            msys2Root = @"C:\msys64";
        }
        string msys2ManPath = Path.Combine(msys2Root, "usr", "share", "man");
        if (Directory.Exists(msys2ManPath))
        {
            paths.Add(msys2ManPath);
        }
    }

    private static void DetectMacOS(List<string> paths)
    {
        AddIfExists(paths, "/usr/share/man");

        // Homebrew
        string? brewPrefix = Environment.GetEnvironmentVariable("HOMEBREW_PREFIX");
        if (brewPrefix is not null)
        {
            AddIfExists(paths, Path.Combine(brewPrefix, "share", "man"));
        }
        else
        {
            AddIfExists(paths, "/opt/homebrew/share/man");
            AddIfExists(paths, "/usr/local/share/man");
        }

        // Xcode command-line tools
        AddIfExists(paths, "/Library/Developer/CommandLineTools/usr/share/man");
    }

    private static void DetectLinux(List<string> paths)
    {
        AddIfExists(paths, "/usr/share/man");
        AddIfExists(paths, "/usr/local/share/man");
    }

    private static void AddIfExists(List<string> paths, string path)
    {
        if (Directory.Exists(path))
        {
            paths.Add(path);
        }
    }

    private static string? FindGitForWindowsInstall()
    {
        // Check common paths first (faster than registry on AOT)
        string[] commonPaths = new[]
        {
            @"C:\Program Files\Git",
            @"C:\Program Files (x86)\Git"
        };

        foreach (string path in commonPaths)
        {
            if (Directory.Exists(path))
            {
                return path;
            }
        }

        // Check PATH for git.exe and resolve its install directory
        string? pathEnv = Environment.GetEnvironmentVariable("PATH");
        if (pathEnv is not null)
        {
            foreach (string dir in pathEnv.Split(Path.PathSeparator))
            {
                string gitExe = Path.Combine(dir, "git.exe");
                if (File.Exists(gitExe))
                {
                    // git.exe is typically in Git/cmd/ — go up one level
                    string? parent = Path.GetDirectoryName(dir);
                    if (parent is not null && Directory.Exists(Path.Combine(parent, "usr", "share", "man")))
                    {
                        return parent;
                    }
                }
            }
        }

        return null;
    }

// src/Winix.Man/PageDiscovery.cs
namespace Winix.Man;

/// <summary>
/// Locates man page files on disk by searching a list of directories
/// in priority order.
/// </summary>
public sealed class PageDiscovery
{
    // Standard man page sections in preference order
    private static readonly int[] SectionSearchOrder = { 1, 8, 6, 2, 3, 4, 5, 7 };

    private readonly IReadOnlyList<string> _searchPaths;

    /// <summary>
    /// Creates a page discovery instance with the given search paths.
    /// Paths are searched in order — first match wins.
    /// </summary>
    public PageDiscovery(IReadOnlyList<string> searchPaths)
    {
        _searchPaths = searchPaths;
    }

    /// <summary>
    /// Resolves the file path for a man page. Returns null if not found.
    /// </summary>
    /// <param name="name">Page name (e.g. "timeit", "grep").</param>
    /// <param name="section">Optional section number. If null, searches all sections.</param>
    public string? FindPage(string name, int? section = null)
    {
        if (section.HasValue)
        {
            return FindInSection(name, section.Value);
        }

        // Search all sections in preference order
        foreach (int sec in SectionSearchOrder)
        {
            string? result = FindInSection(name, sec);
            if (result is not null)
            {
                return result;
            }
        }

        return null;
    }

    /// <summary>
    /// Returns the effective search path list.
    /// </summary>
    public IReadOnlyList<string> GetEffectiveSearchPath()
    {
        return _searchPaths;
    }

    private string? FindInSection(string name, int section)
    {
        foreach (string basePath in _searchPaths)
        {
            string sectionDir = Path.Combine(basePath, $"man{section}");
            if (!Directory.Exists(sectionDir))
            {
                continue;
            }

            // Check uncompressed first, then compressed
            string plainFile = Path.Combine(sectionDir, $"{name}.{section}");
            if (File.Exists(plainFile))
            {
                return plainFile;
            }

            string gzFile = Path.Combine(sectionDir, $"{name}.{section}.gz");
            if (File.Exists(gzFile))
            {
                return gzFile;
            }
        }

        return null;
    }

    /// <summary>
    /// Builds the full search path list: bundled pages, MANPATH entries, and auto-detected paths.
    /// </summary>
    /// <param name="exeDirectory">Directory containing the man binary (for bundled pages).</param>
    /// <param name="manpathEnv">Value of MANPATH environment variable, or null.</param>
    public static IReadOnlyList<string> BuildSearchPaths(string exeDirectory, string? manpathEnv)
    {
        var paths = new List<string>();

        // 1. Bundled pages alongside the binary
        string bundledPath = Path.Combine(exeDirectory, "man");
        if (Directory.Exists(bundledPath))
        {
            paths.Add(bundledPath);
        }

        // 2. MANPATH entries
        if (manpathEnv is not null)
        {
            char separator = Path.PathSeparator; // : on Unix, ; on Windows
            foreach (string entry in manpathEnv.Split(separator, StringSplitOptions.RemoveEmptyEntries))
            {
                string trimmed = entry.Trim();
                if (trimmed.Length > 0 && Directory.Exists(trimmed))
                {
                    paths.Add(trimmed);
                }
            }
        }

        // 3. Auto-detected well-known locations
        foreach (string path in WellKnownPaths.Detect())
        {
            if (!paths.Contains(path))
            {
                paths.Add(path);
            }
        }

        return paths;
    }
}

// src/Winix.Man/BuiltInPager.cs
using System.Text;

namespace Winix.Man;

/// <summary>
/// A minimal built-in pager for interactive display of man pages.
/// Supports scrolling, page up/down, forward search, and quit.
/// Used as a fallback when no external pager is available.
/// </summary>
public sealed class BuiltInPager
{
    /// <summary>
    /// Displays text interactively with scrolling and search.
    /// Blocks until the user quits with 'q'. Restores terminal state on exit.
    /// </summary>
    public void Display(string content)
    {
        string[] lines = content.Split('\n');
        int topLine = 0;
        int termHeight = GetTerminalHeight() - 1; // Reserve one line for status bar
        int termWidth = GetTerminalWidth();
        string? searchTerm = null;
        int searchMatchLine = -1;

        Console.CursorVisible = false;
        try
        {
            Render(lines, topLine, termHeight, termWidth, searchTerm);

            while (true)
            {
                ConsoleKeyInfo key = Console.ReadKey(intercept: true);

                switch (key.Key)
                {
                    case ConsoleKey.Q:
                        return;

                    case ConsoleKey.DownArrow:
                    case ConsoleKey.J:
                        if (topLine < lines.Length - termHeight)
                        {
                            topLine++;
                        }
                        break;

                    case ConsoleKey.UpArrow:
                    case ConsoleKey.K:
                        if (topLine > 0)
                        {
                            topLine--;
                        }
                        break;

                    case ConsoleKey.PageDown:
                    case ConsoleKey.Spacebar:
                        topLine = Math.Min(topLine + termHeight, Math.Max(0, lines.Length - termHeight));
                        break;

                    case ConsoleKey.PageUp:
                        topLine = Math.Max(topLine - termHeight, 0);
                        break;

                    case ConsoleKey.Home:
                        topLine = 0;
                        break;

                    case ConsoleKey.End:
                        topLine = Math.Max(0, lines.Length - termHeight);
                        break;

                    default:
                        if (key.KeyChar == '/')
                        {
                            searchTerm = ReadSearchTerm(termHeight);
                            if (searchTerm is not null && searchTerm.Length > 0)
                            {
                                searchMatchLine = FindNext(lines, topLine + 1, searchTerm);
                                if (searchMatchLine >= 0)
                                {
                                    topLine = Math.Min(searchMatchLine, Math.Max(0, lines.Length - termHeight));
                                }
                            }
                        }
                        else if (key.KeyChar == 'n' && searchTerm is not null)
                        {
                            searchMatchLine = FindNext(lines, topLine + 1, searchTerm);
                            if (searchMatchLine >= 0)
                            {
                                topLine = Math.Min(searchMatchLine, Math.Max(0, lines.Length - termHeight));
                            }
                        }
                        else if (key.KeyChar == 'N' && searchTerm is not null)
                        {
                            searchMatchLine = FindPrevious(lines, topLine - 1, searchTerm);
                            if (searchMatchLine >= 0)
                            {
                                topLine = searchMatchLine;
                            }
                        }
                        break;
                }

                Render(lines, topLine, termHeight, termWidth, searchTerm);
            }
        }
        finally
        {
            Console.CursorVisible = true;
            Console.Clear();
        }
    }

    private static void Render(string[] lines, int topLine, int height, int width, string? searchTerm)
    {
        Console.SetCursorPosition(0, 0);

        for (int i = 0; i < height; i++)
        {
            int lineIndex = topLine + i;
            if (lineIndex < lines.Length)
            {
                string line = lines[lineIndex];
                if (line.Length > width)
                {
                    Console.Write(line[..width]);
                }
                else
                {
                    Console.Write(line);
                    // Clear rest of line
                    int remaining = width - VisibleLength(line);
                    if (remaining > 0)
                    {
                        Console.Write(new string(' ', remaining));
                    }
                }
            }
            else
            {
                Console.Write(new string(' ', width));
            }

            if (i < height - 1)
            {
                Console.WriteLine();
            }
        }

        // Status bar
        Console.WriteLine();
        int percent = lines.Length > 0 ? (int)((topLine + height) * 100.0 / lines.Length) : 100;
        percent = Math.Min(percent, 100);
        string status = searchTerm is not null
            ? $" /{searchTerm}  Line {topLine + 1}/{lines.Length} ({percent}%)"
            : $" Line {topLine + 1}/{lines.Length} ({percent}%)";

        Console.Write("\x1b[7m"); // Reverse video
        Console.Write(status.PadRight(width));
        Console.Write("\x1b[0m");
    }

    private static string? ReadSearchTerm(int statusLine)
    {
        Console.SetCursorPosition(0, statusLine + 1);
        Console.Write("\x1b[7m/");
        Console.Write(new string(' ', GetTerminalWidth() - 1));
        Console.SetCursorPosition(1, statusLine + 1);
        Console.Write("\x1b[0m");
        Console.CursorVisible = true;

        var sb = new StringBuilder();
        while (true)
        {
            ConsoleKeyInfo key = Console.ReadKey(intercept: true);
            if (key.Key == ConsoleKey.Enter)
            {
                break;
            }
            if (key.Key == ConsoleKey.Escape)
            {
                Console.CursorVisible = false;
                return null;
            }
            if (key.Key == ConsoleKey.Backspace && sb.Length > 0)
            {
                sb.Remove(sb.Length - 1, 1);
                Console.SetCursorPosition(1, statusLine + 1);
                Console.Write(sb.ToString() + " ");
                Console.SetCursorPosition(1 + sb.Length, statusLine + 1);
                continue;
            }
            if (key.KeyChar >= ' ')
            {
                sb.Append(key.KeyChar);
                Console.Write(key.KeyChar);
            }
        }

        Console.CursorVisible = false;
        return sb.ToString();
    }

    private static int FindNext(string[] lines, int startLine, string term)
    {
        for (int i = startLine; i < lines.Length; i++)
        {
            if (StripAnsiForSearch(lines[i]).Contains(term, StringComparison.OrdinalIgnoreCase))
            {
                return i;
            }
        }
        // Wrap around
        for (int i = 0; i < startLine; i++)
        {
            if (StripAnsiForSearch(lines[i]).Contains(term, StringComparison.OrdinalIgnoreCase))
            {
                return i;
            }
        }
        return -1;
    }

    private static int FindPrevious(string[] lines, int startLine, string term)
    {
        for (int i = startLine; i >= 0; i--)
        {
            if (StripAnsiForSearch(lines[i]).Contains(term, StringComparison.OrdinalIgnoreCase))
            {
                return i;
            }
        }
        // Wrap around
        for (int i = lines.Length - 1; i > startLine; i--)
        {
            if (StripAnsiForSearch(lines[i]).Contains(term, StringComparison.OrdinalIgnoreCase))
            {
                return i;
            }
        }
        return -1;
    }

    private static string StripAnsiForSearch(string text)
    {
        var sb = new StringBuilder();
        bool inEscape = false;
        for (int i = 0; i < text.Length; i++)
        {
            if (text[i] == '\x1b')
            {
                inEscape = true;
                continue;
            }
            if (inEscape)
            {
                if (text[i] == 'm')
                {
                    inEscape = false;
                }
                continue;
            }
            sb.Append(text[i]);
        }
        return sb.ToString();
    }

    private static int VisibleLength(string text)
    {
        return StripAnsiForSearch(text).Length;
    }

    private static int GetTerminalHeight()
    {
        try { return Console.WindowHeight; }
        catch { return 24; }
    }

    private static int GetTerminalWidth()
    {
        try { return Console.WindowWidth; }
        catch { return 80; }
    }
}

```csharp
// src/Winix.Man/PagerChain.cs
using System.Diagnostics;

namespace Winix.Man;

/// <summary>
/// Determines and invokes the best available pager, searching in priority order:
/// $MANPAGER → $PAGER → sibling winix less → system less → built-in → stdout.
/// </summary>
public sealed class PagerChain
{
    private readonly bool _isTerminal;
    private readonly string _exeDirectory;

    /// <summary>
    /// Creates a pager chain.
    /// </summary>
    /// <param name="isTerminal">Whether stdout is connected to a terminal.</param>
    /// <param name="exeDirectory">Directory containing the man binary (for sibling detection).</param>
    public PagerChain(bool isTerminal, string exeDirectory)
    {
        _isTerminal = isTerminal;
        _exeDirectory = exeDirectory;
    }

    /// <summary>
    /// Sends rendered content through the best available pager.
    /// If stdout is piped, writes directly to stdout.
    /// </summary>
    public void Page(string content, TextWriter stdout)
    {
        if (!_isTerminal)
        {
            stdout.Write(content);
            return;
        }

        // Try external pagers in priority order
        string? pagerCommand = ResolvePager();
        if (pagerCommand is not null)
        {
            PipeToExternalPager(content, pagerCommand);
            return;
        }

        // Fall back to built-in pager
        var pager = new BuiltInPager();
        pager.Display(content);
    }

    private string? ResolvePager()
    {
        // 1. $MANPAGER
        string? manpager = Environment.GetEnvironmentVariable("MANPAGER");
        if (!string.IsNullOrEmpty(manpager))
        {
            return manpager;
        }

        // 2. $PAGER
        string? pager = Environment.GetEnvironmentVariable("PAGER");
        if (!string.IsNullOrEmpty(pager))
        {
            return pager;
        }

        // 3. Sibling winix less
        string siblingLess = Path.Combine(_exeDirectory, "less");
        if (File.Exists(siblingLess))
        {
            return siblingLess;
        }
        string siblingLessExe = Path.Combine(_exeDirectory, "less.exe");
        if (File.Exists(siblingLessExe))
        {
            return siblingLessExe;
        }

        // 4. System less on PATH
        if (IsOnPath("less"))
        {
            return "less";
        }

        return null;
    }

    private static void PipeToExternalPager(string content, string pagerCommand)
    {
        try
        {
            using var process = new Process();
            process.StartInfo.FileName = pagerCommand;
            process.StartInfo.UseShellExecute = false;
            process.StartInfo.RedirectStandardInput = true;
            process.Start();

            process.StandardInput.Write(content);
            process.StandardInput.Close();
            process.WaitForExit();
        }
        catch
        {
            // If pager fails, dump to stdout
            Console.Write(content);
        }
    }

    private static bool IsOnPath(string command)
    {
        string? pathEnv = Environment.GetEnvironmentVariable("PATH");
        if (pathEnv is null)
        {
            return false;
        }

        string extension = OperatingSystem.IsWindows() ? ".exe" : "";
        foreach (string dir in pathEnv.Split(Path.PathSeparator))
        {
            string fullPath = Path.Combine(dir, command + extension);
            if (File.Exists(fullPath))
            {
                return true;
            }
        }

        return false;
    }
}

// src/man/Program.cs
using System.Reflection;
using Winix.Man;
using Yort.ShellKit;

namespace Man;

internal sealed class Program
{
    static int Main(string[] args)
    {
        string version = GetVersion();

        var parser = new CommandLineParser("man", version)
            .Description("Display manual pages. Renders groff man pages to the terminal with colour and hyperlinks.")
            .StandardFlags()
            .Flag("--no-pager", "Write to stdout without paging")
            .IntOption("--width", null, "N", "Override rendering width (default: terminal width, max 80)",
                n => n < 10 ? "must be >= 10" : null)
            .Flag("--path", "-w", "Print file path to man page instead of rendering")
            .Flag("--where", "Alias for --path (GNU man compatibility)")
            .Flag("--manpath", "Print the effective search path and exit")
            .Positional("[[section] page]")
            .ExitCodes(
                (ManExitCode.Success, "Page found and displayed"),
                (ManExitCode.NotFound, "Page not found"),
                (ManExitCode.UsageError, "Usage error"),
                (ExitCode.UsageError, "Internal tool error"),
                (ExitCode.NotExecutable, "Cannot execute pager"),
                (ExitCode.NotFound, "Required dependency not found"));

        var result = parser.Parse(args);
        if (result.IsHandled) { return result.ExitCode; }
        if (result.HasErrors) { return result.WriteErrors(Console.Error); }

        // Resolve options
        bool useColor = result.ResolveColor();
        bool isTerminal = ConsoleEnv.IsTerminal(checkStdErr: false);
        bool noPager = result.Has("--no-pager");
        bool showPath = result.Has("--path") || result.Has("--where");
        bool showManpath = result.Has("--manpath");

        // Width: --width flag > MANWIDTH env var > terminal width (capped at 80)
        int? widthOverride = null;
        if (result.Has("--width"))
        {
            widthOverride = result.GetInt("--width");
        }
        else
        {
            string? manwidth = Environment.GetEnvironmentVariable("MANWIDTH");
            if (manwidth is not null && int.TryParse(manwidth, out int mw) && mw >= 10)
            {
                widthOverride = mw;
            }
        }

        // Build search paths
        string exeDir = Path.GetDirectoryName(Environment.ProcessPath) ?? ".";
        string? manpathEnv = Environment.GetEnvironmentVariable("MANPATH");
        IReadOnlyList<string> searchPaths = PageDiscovery.BuildSearchPaths(exeDir, manpathEnv);
        var discovery = new PageDiscovery(searchPaths);

        // Handle --manpath
        if (showManpath)
        {
            foreach (string path in discovery.GetEffectiveSearchPath())
            {
                Console.WriteLine(path);
            }
            return ManExitCode.Success;
        }

        // Parse positional args: [section] page
        string[] positionals = result.Positionals;
        if (positionals.Length == 0)
        {
            Console.Error.WriteLine("man: what manual page do you want?");
            return ManExitCode.UsageError;
        }

        string pageName;
        int? section = null;

        if (positionals.Length >= 2 && int.TryParse(positionals[0], out int sec))
        {
            section = sec;
            pageName = positionals[1];
        }
        else
        {
            pageName = positionals[0];
        }

        // Find the page
        string? filePath = discovery.FindPage(pageName, section);
        if (filePath is null)
        {
            Console.Error.WriteLine($"man: no manual entry for {pageName}" +
                (section.HasValue ? $" in section {section}" : ""));
            return ManExitCode.NotFound;
        }

        // Handle --path
        if (showPath)
        {
            Console.WriteLine(filePath);
            return ManExitCode.Success;
        }

        // Handle --json
        if (result.Has("--json"))
        {
            // Quick metadata extraction — parse just the TH line
            string raw = ManPageFileReader.Read(filePath);
            var lexer = new GroffLexer();
            var expander = new ManMacroExpander();
            var blocks = expander.Expand(lexer.Tokenise(raw));
            var title = blocks.OfType<TitleBlock>().FirstOrDefault();

            // Extract description from NAME section
            string description = ExtractDescription(blocks);

            Console.Error.WriteLine(FormatJson(
                pageName, title, description, filePath, version));
            return ManExitCode.Success;
        }

        // Read and render
        string source = ManPageFileReader.Read(filePath);

        var groffLexer = new GroffLexer();
        var macroExpander = new ManMacroExpander();
        var rendererOptions = new RendererOptions
        {
            WidthOverride = widthOverride,
            Color = useColor,
            Hyperlinks = useColor && isTerminal // OSC 8 only when colour is on and interactive
        };
        var renderer = new TerminalRenderer(rendererOptions);

        IReadOnlyList<DocumentBlock> documentBlocks = macroExpander.Expand(groffLexer.Tokenise(source));
        string rendered = renderer.Render(documentBlocks);

        // Page the output
        if (noPager || !isTerminal)
        {
            Console.Write(rendered);
        }
        else
        {
            var pagerChain = new PagerChain(isTerminal, exeDir);
            pagerChain.Page(rendered, Console.Out);
        }

        return ManExitCode.Success;
    }

    private static string ExtractDescription(IReadOnlyList<DocumentBlock> blocks)
    {
        // Look for the paragraph after the NAME section heading
        bool inNameSection = false;
        foreach (DocumentBlock block in blocks)
        {
            if (block is SectionHeading sh && sh.Text == "NAME")
            {
                inNameSection = true;
                continue;
            }

            if (inNameSection && block is Paragraph para)
            {
                string text = string.Join("", para.Content.Select(s => s.Text));
                // Format is typically "name \- description"
                int dashIndex = text.IndexOf(" - ", StringComparison.Ordinal);
                if (dashIndex < 0)
                {
                    dashIndex = text.IndexOf("- ", StringComparison.Ordinal);
                }
                return dashIndex >= 0 ? text[(dashIndex + 2)..].Trim() : text.Trim();
            }

            if (inNameSection && block is SectionHeading)
            {
                break;
            }
        }
        return "";
    }

    private static string FormatJson(
        string name, TitleBlock? title, string description, string filePath, string version)
    {
        string section = title?.Section ?? "";
        string titleName = title?.Name ?? name.ToUpperInvariant();
        string date = title?.Date ?? "";
        string source = title?.Source ?? "";

        return $"{{\"tool\":\"man\",\"version\":\"{Escape(version)}\",\"exit_code\":0," +
            $"\"name\":\"{Escape(name)}\",\"section\":{(section.Length > 0 ? section : "null")}," +
            $"\"title\":\"{Escape(titleName)}\",\"description\":\"{Escape(description)}\"," +
            $"\"source\":\"{Escape(source)}\",\"date\":\"{Escape(date)}\"," +
            $"\"file_path\":\"{Escape(filePath.Replace("\\", "/"))}\"}}";
    }

    private static string Escape(string value)
    {
        return value.Replace("\\", "\\\\").Replace("\"", "\\\"").Replace("\n", "\\n").Replace("\r", "\\r");
    }

    private static string GetVersion()
    {
        return typeof(ManExitCode).Assembly
            .GetCustomAttribute<AssemblyInformationalVersionAttribute>()
            ?.InformationalVersion ?? "0.0.0";
    }
}

# man

Cross-platform man page viewer with colour and hyperlinks. Renders groff man pages natively — no groff installation required.

## Install

### .NET tool (requires .NET SDK)

```bash
dotnet tool install -g Winix.Man
```

### Standalone binary

Download from [GitHub Releases](https://github.com/Yortw/winix/releases).

### Scoop (Windows)

```powershell
scoop bucket add winix https://github.com/Yortw/winix
scoop install man
```

## Usage

```bash
man timeit              # show timeit(1)
man 3 printf            # show printf(3) specifically
man --path timeit       # print file path, don't render
man --manpath           # show search path
man --no-pager timeit   # dump to stdout, no paging
```

## Options

| Flag | Short | Description |
|------|-------|-------------|
| `--no-pager` | | Write to stdout without paging |
| `--color` | | Force coloured output |
| `--no-color` | | Disable coloured output |
| `--width N` | | Override rendering width |
| `--path` | `-w` | Print file path instead of rendering |
| `--where` | | Alias for `--path` |
| `--manpath` | | Print effective search path |
| `--json` | | Output page metadata as JSON |
| `--help` | `-h` | Show help |
| `--version` | | Show version |

## Page Discovery

Pages are searched in this order:

1. **Bundled pages** — shipped alongside the binary
2. **`MANPATH`** — environment variable (`;` on Windows, `:` on Unix)
3. **Auto-detected** — Git for Windows, MSYS2, Homebrew, `/usr/share/man`

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Page found and displayed |
| 1 | Page not found |
| 2 | Usage error |
| 125 | Internal tool error |

## Colour

When connected to a terminal, man uses ANSI colour for section headings and bold/underline for styled text. Colour is disabled automatically when piped. Override with `--color` / `--no-color` or the `NO_COLOR` environment variable.
